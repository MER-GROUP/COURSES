Напомним, что турниром T называется орграф, 
любые две вершины которого соединены ровно одним ориентированным ребром. 
Иными словами, турниром называется орграф, полученный из полного графа K_n
​произвольной ориентацией его ребер.

Доказать, что в любом турнире T существует гамильтонов путь.

---------------
Поскольку любые две вершины произвольного  турнира  смежны, то существует цепь, 
содержащая все вершины турнира.
---------------
Доказательство проведём индукцией (вы ждали, и вы получили, как говорится)

1) Для турнира построенного на 1 вершине гамильтонов путь, очевидно, существует.

2) Пусть в турнире, построенном на n - 1 вершине, существует гамильтонов путь

3) Докажем существование для n вершин:

Для начала, удалим одну из вершин графа (назовем её  X), получим турнир, построенный на n-1 вершине. В нём по предположению (2) есть гамильтонов путь. Обозначим его вершины по порядку x1->.->.->.->xn-1. 

Существует три варианта

1) В удаленную вершину входит ребро из xn-1, тогда мы можем продолжить искомый путь через ребро xn-1 -> X

2) Из удаленной вершины выходит ребро в x1, тогда мы можем продолжить искомый путь через ребро X -> x1

3) Приведенных выше ребер нет, но есть ребро xi-1 -> X и ребро X -> xi (именно с такими направлениями), тогда нужный путь получится заменой xi-1->xi на xi-1->X->xi

То есть, в турнире на n вершинах существует гамильтонов путь

Таким образом, утверждение доказано
---------------
докажем что если в турнире размером n есть гамильтонов путь,то в турнире n+1 тоже есть гамильтонов путь. Вершина n+1 соединина с остальными вершинами n ребрами, пусть l ребервыходят из вершины n+1 и входят в l вершин в гамильтоновом пути. если l=0, то все ребра входят в вершину n+1 и гамильтонов путь можно продолжить просто пройдя из n в n+1. если l=n то все вершины выходят из вершины n+1 и можнопродолжить путь начав его из вершины n+1 и перейдя в вершину 1. Если l!=0 или n, и из вершины n нельзя перейти в n+1 и из n+1 в 1,то возможно найти две соседние вершины i из которой выходт ребро в n+1и i+1 в которую входит ребро из n+1. Тогда можно дойти до вершины i по первоначальному гамильтонову пути перейти по выходящему ребру до n+1 и перейти в вершину i+1 и продолжить первичный гамильтонов путь. в турнире из двух вершин есть гамильтонов путь. тогда в любом турнире есть гамильтонов путь
